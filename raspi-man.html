固定IPアドレスの設定

1️⃣ ラズパイにモニタ及びキーボードとマウスを接続します。

2️⃣ ラズパイのデスクトップ画面が表示されたら左上の「ラズパイアイコン」→「アクセサリ」→「LXTerminal」をクリックしてターミナルを起動します。

3️⃣ ターミナルから下記コマンドで"dhcpcd.conf"ファイルを開いて、最下行付近の"interface eth0"以下のIPアドレス部分を編集します。

$ sudo nano /etc/dhcpcd.conf

.....
interface wlan0
static ip_address=192.168.1.206/24
static routers=192.168.1.1
static domain_name_servers=192.168.1.1 8.8.8.8
interface eth0
static ip_address=192.168.1.120
static routers=192.168.1.1
static domain_name_servers=192.168.1.1 8.8.8.8
  • interface ⇒ eth0
  • static ip_address ⇒ 固定IPアドレス
  • static routers ⇒ デフォルトゲートウェイのIPアドレス
  • static domain_name_servers ⇒ DNSサーバーのIPアドレス

4️⃣ ctrl+oで上書き保存、ctrl+xでnanoエディタを終了します。

5️⃣ 下記コマンドで再起動すると設定が有効になります。

$ sudo reboot
※ 再起動後 "ifconfig"コマンドで設定したIPアドレスになっていることを確認してください。
